# Trip Planner for Truck Drivers

A full-stack web application that helps truck drivers plan trips efficiently by generating routes, fuel stops, rest stops, and driver ELD logs.

The system visualizes trips on a map and displays driver logs through charts, similar to real fleet management software used in companies.
